IMPORTANTE
code .   ===> Quando você escreve "code ." dentro do terminal do PowerShell, você está usando um comando específico para iniciar o Visual Studio Code no diretório atual. O ponto "." representa o diretório atual. Portanto, o comando "code ." instrui o PowerShell a abrir o Visual Studio Code com o diretório atual como o diretório de trabalho. Isso é útil quando você deseja trabalhar em um projeto específico e deseja abrir o Visual Studio Code nesse contexto.
